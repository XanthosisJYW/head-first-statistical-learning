# 统计学习初步 Demo 说明

## 运行环境 & 依赖

本 Demo 需要 Python3 的运行环境, 并依赖如下第三方库:

- numpy
- matplotlib
- pandas
- scikit-learn
- statsmodels
- pydotplus

建议先搭建虚拟环境, 后安装依赖, 避免污染既有项目.

## 项目结构

- `example` 目录下的内容即 Demo 文件, 主要有
    - linear_regression.py, 线性回归
    - decision_tree.py, 决策树
    - pca.py, 主成分分析
- `data` 目录下的内容为原始数据
